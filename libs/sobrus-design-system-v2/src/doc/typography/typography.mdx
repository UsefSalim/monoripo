
# Typography

## Paragraph

Paragraphs are the base component for body content, crafted for readability in longer text bodies.

---

### Token: `font-paragraph/large`

**Font-Family:** Inter

| Value         | Variable                    |
|---------------|-----------------------------|
| Font-Size     | 18px - 1.125rem             | `--font-size-4`        |
| Font-Weight   | 400                         | `--font-weight-regular`|
| Line-Height   | 22px - 1.6875rem            | `--font-line-height-0` |

Example Text:

> We ignite opportunity by setting the world in motion.  
> 0123456789
